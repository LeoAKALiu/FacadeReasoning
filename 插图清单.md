# 软件使用说明书 插图清单

## 自动截图（推荐）

无需手动截图。请按以下步骤生成配图并生成带插图的 Word：

1. **启动应用**（二选一）  
   - 本地：`npm run dev`，默认地址 `http://localhost:3000`  
   - 或使用已部署地址，见下。

2. **自动截取 38 张图**（脚本使用 Puppeteer 自带的 Chromium）  
   - 若截图里**中文显示为方框**：在运行截图的 Linux 服务器上安装中文字体后重跑，例如：  
     `sudo yum install -y fontconfig wqy-zenhei-fonts && fc-cache -fv`  
   - 脚本也会自动注入网页中文字体（Noto Sans SC）作为后备。  
   ```bash
   # 本地
   BASE_URL=http://localhost:3000 npm run screenshots
   # 或使用部署地址
   BASE_URL=http://47.98.176.41:3001 npm run screenshots
   ```  
   截图会保存到项目根目录 `screenshots/`（图1.png～图38.png）。

3. **生成 Word（自动嵌入截图）**  
   ```bash
   npm run manual:docx
   # 或：python3 scripts/gen_manual_docx.py
   ```  
   若存在 `screenshots/图N.png`，说明书中会自动嵌入对应图片；缺失的图保留占位框。

**系统基址**：`http://47.98.176.41:3001/`（或本地 `http://localhost:3000`）

# Case assets

Place real assets per case in `case-01/`, `case-02/`, `case-03/` with these names (replace placeholders as needed):

- `facade-original.jpg` — original facade photo
- `facade-annotated.png` — evidence overlay
- `facade-rectified.png` — rectified image
- `detail-01.png` … `detail-04.png` — detail crops
- `structure-sketch.svg` — structure sketch (placeholder provided)
- `parameter-preview.png` or `.svg` — parameter table preview (placeholder provided)
- `component-candidates.svg` — component candidates (placeholder provided for case-02, case-03)
- `report-preview.png` — report preview (optional)

If a file is missing, the app falls back to inline placeholders.
